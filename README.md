#  Cryptobeam Investor Portal

**Secure Accredited Investor Gateway for the Cryptobeam Fund**

A private investor portal built on **Webflow + Auth0**, enabling accredited investor onboarding, NDA execution, and secure access to fund documents.

##  Features
- Accreditation verification (Reg D 506(b))
- Digital NDA execution (DocuSign)
- Encrypted access to fund documents
- Subscription form for limited partnership commitments
- CRM and compliance team notification workflow

##  Documents Available
- AML Compliance Statement
- Limited Partnership Agreement (LPA)
- SEC Form D Template
- Private Placement Memorandum (PPM)
- Subscription Agreement

##  Tech Stack
Webflow 路 Auth0 路 Firebase 路 DocuSign API 路 Plaid

## Л Compliance Alignment
Follows **SEC Reg D Rule 506(b)** and **AML/KYC** protocols for accredited investors.
